About Very Useful Map


                프로젝트 명 : 매우 유용한 지도(가제)

                목적 : 학교 제휴 매장 확인 및 근처 식당 탐색 - 세종대학교 학생들을 위한 웹

                주요 기능 : 지도를 통한 주변 식당 확인, 식당 메뉴 종류를 필터링, 순위를 매겨 제공, 룰렛 기능 등

                사용 언어 : HTML, CSS, JS
                데이터베이스 : MySQL, MariaDB

                학습에 대한 목적 : 웹사이트의 구성 요소에 대한 고찰 및 실습, API를 활용한 실전형 프로그래밍
